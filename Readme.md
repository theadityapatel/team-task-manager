# TaskFlow - Team Task Manager

A full-stack task management web application built using React, Node.js, and MongoDB.

## 🚀 Features

- User Signup & Login (JWT Authentication)
- Role-based system (Admin / Member)
- Create, update, delete tasks
- Dashboard with task tracking
- Profile management
- Clean UI with sidebar navigation

## 🛠 Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express

Database:
- MongoDB Atlas

Authentication:
- JWT

## 📦 Installation

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev  

## 🔐 Environment Variables

Create `.env` file in backend:

MONGO_URI=your_mongodb_uri  
JWT_SECRET=your_secret_key
